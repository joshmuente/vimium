Vimium Search Completion

Search completion is available for custom search engines whose search URL matches one of Vimium's built-in completion engines; that is, the search URL matches one of the regular expressions below. Search completion is not available for the default search engine.

Custom search engines can be configured on the options page.
Further information is available on the wiki.
